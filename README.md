# Twitter (X) Clone

A responsive Twitter (X) clone built with HTML and Tailwind CSS to practice utility-first CSS and modern frontend development.

## ✨ Features
- Responsive design
- Sidebar navigation
- Feed section
- Trending section
- Modern Twitter (X) inspired UI

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser

## 📌 Learning Outcome

This build helped me strengthen my understanding of Tailwind CSS, responsive layouts, and utility-first styling.
